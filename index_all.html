Как проверить бонусный баланс

Карта почётного клиента в формате Passbook автоматически
обновляется на следующий день после совершения вами покупки.
Обратите внимание, что бонусный балланс отображает общее
количество бонусов. По условиям программы активация бонусов
после покупки происходит в течение 14 календарных дней.
То есть использовать их как скидку для новых покупок можно
только по истечении этого периода.

Как проверить бонусный баланс

В настоящее временя на платформе Windows Phone есть
ограничение по поддержке электронных карт в формате Passbook.
Есть платное приложение Wallet Pass, которое может отображать баланс в момент установки карты на смартфон, но не может автоматически его обновлять.

  • – Вы можете установить из магазина Windows Phone
    приложение Wallet Pass (приложение платное).
  • – Откройте эту страницу в браузере на смартфоне.
  • – Коснитесь значка «Добавить в Passbook».
  • – Выберите в списке приложений Wallet Pass.
  • – Коснитесь значка с изображением дискеты
    в нижней части экрана, чтобы сохранить карту.
Подробная инструкция по тому, как загружать
приложения из магазина Windows Phone.
